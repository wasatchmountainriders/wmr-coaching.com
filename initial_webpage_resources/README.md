# Wasatch Mountain Riders — Coach Resources

Self-contained HTML coaching resources for the WMR youth XC mountain biking program. All files work offline — no server, no installation needed. Just open `index.html` in any web browser.

## Start here

**`index.html`** — Landing page. Open this first. Links to all 7 program decks and the universal reference.

## The 7 program decks

Each deck has 12 skill cards covering a 10-week program. Editorial design optimized for pre-season reading on phone, tablet, or desktop.

- **`lil_shredders_skill_cards.html`** — Ages 5–7 (coed)
- **`beginner_boys_skill_cards.html`** — Ages 7–9
- **`beginner_girls_skill_cards.html`** — Ages 7–9
- **`intermediate_boys_skill_cards.html`** — Ages 8–11
- **`intermediate_girls_skill_cards.html`** — Ages 8–11
- **`advanced_boys_skill_cards.html`** — Ages 9–12
- **`advanced_girls_skill_cards.html`** — Ages 9–12

## Universal reference

**`reference.html`** — Read once at the start of the season. Reference whenever something goes sideways. Four sections:

- **Protocols** — Crash protocol, head injuries, weather, lost rider, communication chain
- **Stuck** — Meltdowns, refusals, fear, conflict, when you don't know what to do
- **Readiness** — M-Check, helmet fit, hydration, head count, coach kit, pre-ride briefing
- **Standards** — Coach conduct, body comments, one-on-one rule, hands-off rule, photography, mandated reporting, required training

## How to use

**Before the season:**
1. Open `index.html`
2. Read your program's deck cover-to-cover
3. Read the reference page cover-to-cover
4. Sign up for Utah League OTB101 training (free): contact Ben at utahmtb.org

**During the season:**
1. Before each session, re-read the cards for the skills you're teaching that day
2. When something goes sideways, open the reference page
3. Trust your training. Trust your gut. Trust your coaches.

## Notes

- All files are self-contained — no dependencies, no internet required after first load (fonts cached).
- Use arrow keys to navigate between cards within a deck.
- Print-friendly: each card prints on its own page.
- Mobile-optimized: works on any phone.
